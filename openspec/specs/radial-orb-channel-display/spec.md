# radial-orb-channel-display Specification

## Purpose

**Superseded.** The 16-channel meter visualization is defined by **`sphere-3d-channel-display`** (3D sphere, orthographic orbit, HSL lightness). The former 2D donut and opacity-only requirements are **removed** from the product.

## Requirements

_There are no active requirements in this capability._ See `openspec/specs/sphere-3d-channel-display/spec.md`.
